Example Config

{ "class": "contactform", "name": "contactform", "label": "Contact Form", "delivery-template": "email_template.html", "delivery-to": "John Smith ", "delivery-subject": "Website Enquiry", "confirmation-template": "confirmation_template.html", "confirmation-replyto": "John Smith ", "confirmation-subject": "Confirmation email from ", "success-address": "?success=true", "check-for-spam": "off" }
Configuration Option Valid Values
id Unqiue name with no spaces or special characters
delivery-template HTML File rendered to send business owner
delivery-to Submit contact form to this email address. For example the business owner.
delivery-subject Email subject
confirmation-template

HTML File rendered to send recipient.

When this field is provided the script will automatically look for a field called "email" and send this rendered template to enquirer.

confirmation-replyto Provides an email which form filler can respond back to.
confirmation-subject Email subject
success-address When form submits returns user to the success page
check-for-spam Turns spam filter on and off
