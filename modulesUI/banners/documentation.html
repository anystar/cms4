Banners

Quick use

Place the link tag in the header of the html file.
Place the script tag just before the body's end tag.
Place the template variable {{@banners.html}} where you want the banner to display.

How to works

The banner system is located in a single folder under uploads. It contains three important files that contains the css, javascript and html for the banner system. Traditionally banner systems will require a stylesheet, a javascript file, some javascript to initilize the banner with correct settings and the html to list out all the images for the banner system to use. Generally in an unordered list.

Here we have merged the core javascript file and initlization code into one file. This is located under clients upload folder as slider.js.
