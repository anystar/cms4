How the module works

The dependant files for this module are located in a single folder in the clients directory under uploads. This folder contains the banner images plus three text files. A script file, stylesheet file and a html file for the mark up.

  • slider.css - Contains stylesheet
  • slider.js - Contains core javascript and initilize code
  • slider.html - Contains html and template code to structure markup

Traditionally banner systems will require a stylesheet, a javascript file, some javascript to initilize the banner with correct settings and the html markup to list out all the images for the banner system to use. Generally the html markup is a simple unordered list.

In this module we have merged the core javascript file and initlization code into a single file. This is located under clients upload folder as slider.js. We have also put the css in a slider.css and html markup in slider.html. You can edit these files directly through the admin panel or through a text editor. Nothing special happens through the admin panel, all it does is replaces the contents of each file.

You do not need to follow the design of this module. The admin panel simply gives a convenient way to demo and view the files. Feel free to ignore the dependant files and use it however you like. But if you want it to demo in the admin panel please use the dependant files.

Quick use

Place the link tag in the header of the html file. 

<link href="{{@BASE}}/banners/slider.css" rel="stylesheet">

Place the script tag just before the body's end tag. 
<script src="{{@BASE}}/banners/slider.js"></script>

Place the template variable {{@banners.html}} where you want the banner to display.

Advanced use

If you need more control you can build the html yourself with the template code.

For example: 

<repeat group="{{@banners.images}}" value="{{@image}}">
	<img src="{{@BASE}}/{{@image.url}}" />
</repeat>

Template code

Template Variable Example
{{@banners.html}} Contains compiled html markup
{{@banners.images}} An array of images
{{@banners.images[x].url}} Address of image.
Eg: <img src="{{@BASE}}/{{@image.url}}" />
{{@banners.images[x].filename}} Just the file name
