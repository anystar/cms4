Quick use

Place the template variable {{@contact.html}} where you want the contact form to display.

Advanced use

If you need more control you can build the html yourself with the template code.

For example: 

<repeat group="{{@banners.images}}" value="{{@image}}">
	<img src="{{@BASE}}/{{@image.url}}" />
</repeat>

Template code

Template Variable Example
{{@contact.html}} Contains compiled html markup of contact form
{{@contact.form}} An array of form fields
{{@contact.form[ i ].id}} ...
{{@contact.form[ i ].label}} ...
{{@contact.form[ i ].value}} ...
{{@contact.form[ i ].error}} ...
{{@contact.form[ i ].type}} ...

Database

The order of the images is stored in the settings table as a json string. Nothing else is stored. This means that deleting the "slider" folder will essentially delete the module.
