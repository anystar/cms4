Creating a new CMS

Start by building the website. Feel free to structure it however you like.

Once you're ready create a file called "cms.php" and include the code below in it. 

	<?php
	require "/home/cms/boot.php";
Change "/home/cms/boot.php" to the path where the cms is located. For example a windows machine may be "C:\\webserver\cms\boot.php".

In a web browser go to http://WEBSITE.com/cms.php

The cms.php script will execute once creating any necessary folders and files. Afterwards you'll be redirected to the admin panel.

You can now login with to the cms with the webmaster user and pass.

If any issue should occur error messages will be displayed letting you know what went wrong.

Installing Web Works CMS on a new server

Web Works CMS requires a handful of dependencies.

  • Apache with mod_rewrite, sqlite, GD and PHP.
  • Fat Free Framework
  • ckeditor

The primary location that the CMS can be downloaded from is on the web works server.

/home/cms/stable or /home/cms/experimental

The folder containing the CMS can be downloaded as is. For example one may download the stable version via FTP.

It is recommended to use git to download the cms to maintain the source code. One may download the stable version via git like this:- 

git clone root@webworksau.com:/home/cms/stable .
