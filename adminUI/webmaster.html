Creating a new CMS

Start by building the website. Feel free to lay it out however you like.

Once your ready simple create a file called "cms.php" and include the code below. 

	<?php
	require_once("/home/cms/boot.php");
Now browse to cms.php and it will automatically create any files or folders it needs. It'll also give you error messages if it requires something that is missing or if something is mis-configured.
